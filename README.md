## Getting Started

Docker composer lemp stack include php custom extensions

### Prerequisites

What things you need to install the software and how to install them

```
sudo pip install docker-compose
```

### How to use

Up
```
docker-compose up -d
```

Down

```
docker-compose down
```
View Logs

```
docker logs lemp_nginx
```
